# Installation
You will need `node` and `npm` installed globally on your machine.

Clone down this repository:
`gitclone https://github.com/eytanbab/react/tree/master/url-shortener-2`

Go into the repository:
`cd url-shortener-2`

Install dependencies:
`npm install`

Run the webapp:
`npm run dev`
