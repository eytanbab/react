# Installation
You will need `node` and `npm` installed globally on your machine.

Clone down this repository:
`gitclone https://github.com/eytanbab/react/tree/master/weather-web-app`

Go into the repository:
`cd weather-web-app`

Install dependencies:
`npm install`

Run the webapp:
`npm run dev`
