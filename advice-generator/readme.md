# Installation
You will need `node` and `npm` installed globally on your machine.

Clone down this repository:
   `gitclone https://github.com/eytanbab/react/tree/master/advice-generator`

Go into the repository:
`cd advice-generator`

Install dependencies:
`npm install`

Run the webapp:
`npm run dev`
